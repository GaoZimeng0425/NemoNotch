# i18n 国际化设计方案

## 概述

为 NemoNotch 添加国际化支持，第一步支持简体中文 + English，应用内可切换语言。

## 技术方案

### 核心组件

| 组件 | 说明 |
|------|------|
| `Localizable.xcstrings` | String Catalog，统一管理所有翻译 |
| `AppLanguage` enum | 语言选项：system / en / zh-Hans |
| `.environment(\.locale:)` | 在根视图注入 locale，SwiftUI Text 自动查找 |

## 实现步骤

### Step 1: 创建 AppLanguage enum + 更新 AppSettings

在 `Models/AppSettings.swift` 中：

```swift
enum AppLanguage: String, CaseIterable, Codable {
    case system
    case en
    case zhHans = "zh-Hans"

    var locale: Locale? {
        switch self {
        case .system: nil
        case .en: Locale(identifier: "en")
        case .zhHans: Locale(identifier: "zh-Hans")
        }
    }

    var displayName: String {
        switch self {
        case .system: String(localized: "settings.language.system")
        case .en: "English"
        case .zhHans: "简体中文"
        }
    }
}
```

AppSettings 增加：
```swift
var language: AppLanguage {
    didSet { UserDefaults.standard.set(language.rawValue, forKey: "language") }
}
```

### Step 2: 创建 Localizable.xcstrings

在 Xcode 项目中添加 String Catalog 文件，填入上述所有 key 和翻译。

### Step 3: 在三个根视图注入 locale

**NotchView（NotchCoordinator 创建时）**:
```swift
NotchView()
    .environment(\.locale, settings.language.locale ?? Locale.current)
```

**SettingsView（showSettings 时）**:
```swift
SettingsView()
    .environment(\.locale, settings.language.locale ?? Locale.current)
```

**MenuContent（MenuBarExtra）**:
```swift
MenuContent(...)
    .environment(\.locale, ...)
```

需要在 NemoNotchApp 中访问 AppSettings，通过 AppDelegate 暴露。

### Step 4: 替换所有硬编码字符串

17 个文件，87 个字符串，逐一替换 `Text("中文")` → `Text("key")`。

### Step 5: Settings 增加语言选择 UI

在 SettingsView 添加语言选择 section。

### Step 6: 更新 README

同步更新 README.md 和 README_CN.md 的功能描述。
